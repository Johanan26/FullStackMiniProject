# Full Stack Project

## Features

- **Firebase Authentication** - Login, signup, and session management
- **User Profiles** - Avatar upload, name, and age management
- **Social Media Posts** - Twitter/Instagram-style feed with post creation, viewing, editing, and deletion
- **Post Ownership** - Users can only edit/delete their own posts
- **Dark Theme** - Modern dark theme throughout the application
- **MongoDB Integration** - All data stored in MongoDB
- **Command Architecture** - Clean, modular code structure

## Setup

1. Install dependencies:
   ```bash
   cd next/fullStackWeek11lab2 && npm install
   cd ../node/microservices && npm install
   ```

2. Start MongoDB (localhost:27017)

3. Start backend:
   ```bash
   cd node/microservices && npm start
   ```

4. Start frontend:
   ```bash
   cd next/fullStackWeek11lab2 && npm run dev
   ```

## Tech Stack

- Next.js + React (Frontend)
- Node.js + Express (Backend)
- MongoDB (Database)
- Firebase Auth (Authentication)
